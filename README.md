# 🏢 Gestión Laboral

Sistema web de gestión laboral orientado a la administración eficiente de empleados, asistencias y procesos internos empresariales.

---

## 🚀 Vista general

**Gestión Laboral** permite centralizar la información del personal y automatizar tareas administrativas, mejorando la organización y reduciendo errores operativos.

---

## ✨ Funcionalidades

### 👑 Administrador

* Gestión de usuarios, áreas y puestos
* Control de asistencias, ausencias y licencias
* Configuración de deducciones y adicionales
* Generación y administración de recibos
* Auditoría de acciones del sistema
* Gestión de cierres mensuales

### 👤 Empleado

* Registro de jornada laboral
* Consulta de recibos
* Historial de actividad

---

## 🧱 Arquitectura

El proyecto está organizado bajo una **arquitectura modular por dominio**, lo que permite escalabilidad y mantenimiento eficiente.

```id="arch12"
js/
 ├── modules/
 │    ├── admin/
 │    └── empleado/
 ├── utils/
 ├── config.js
 └── app.js
```
